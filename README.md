## 🎤 システム概要

簡単なメモ帳です

## 🎤 機能

- [ ] 背景色を変更
- [x] メモの記入
- [ ] 編集
- [x] データ保存

## 🎤 使用環境

- macOS
- Xcode

## 🎤 インストール

```
$ git clone https://github.com/〜〜 //クローンする
$ cd 〜〜 //クローンしたディレクトリに移動する
```

## 🎤 使い方

1. 「＋」：メモ内容を追加できます
2. 「タスクを選択」：🟥 未実施から ✅ 実施済みに変更
3. 「削除」：実施済みの内容を削除
